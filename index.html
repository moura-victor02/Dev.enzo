HTML5

O HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Ou seja, ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos. Portanto, ele serve para dar significado e organizar as informações de uma página na web.

CSS3

O CSS é uma linguagem de folha de estilo composta por “camadas”, criado com o propósito de estilizar as páginas HTML, ou seja, definir a aparência das páginas, para deixá-las visualmente mais bonitas e agradáveis, podendo alterar a fonte, cor, posicionamento dos elementos, layout e muito mais.

JS

O JS (JavaScript) é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma (protótipos, orientado a objeto, imperativo e funcional). Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

VUE.js

O Vue.js é um framework JS open source conhecido pela sua reatividade. Usado para construir SPA e interfaces de usuário, possui componentes reutilizáveis e proporciona um desenvolvimento ágil. Ao contrário de outros frameworks monolíticos, Vue foi projetado desde sua concepção para ser adotável incrementalmente.
